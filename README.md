# My cypress automation

This project utilizes Cypres to automate the testing of a login feature and user update feature.
Cypress is a Javascript-based end-to-end testing framework that allows developers to write tests and execute them in a real browser.

## Instalation
```bash
npm install
```
> **NOTE**
>
> It is necessary to have **node** installed 

## Running
```bash
# You open the Cypress UI
npx cypress open

# Run the automation bycommand line
npx cypress run
```
